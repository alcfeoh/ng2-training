Lab A0: Dev environment set-up

  1. Go to your labs folder: C:\sources\ng2-training-master
  2. Open a terminal in that folder and run ng serve. This command compiles the code and starts a local server on your machine
  3. Open http://localhost:4200 in your browser to see the result
  4. You should see Hello Angular displayed in your browser


Note: We use Angular CLI (https://cli.angular.io) to build & serve our code locally.

Lab C1: Hello World

  1. Open your IDE and use C:\sources\ng2-training-master as your main project location
  2. Open the file src/app/hello/hello.component.ts in your IDE
  3. Add a class property firstName = 'John' to the HelloComponent class.
  4. In the @Component decorator, edit the template property to add a new expression firstName in the HTML template.
  5. Your file should look like this: 
    import { Component } from '@angular/core';

@Component({
  selector: 'app-hello',
  template: `
    <div>
      <h2>Hello {{ '{{'}}firstName}} {{ '{{'}}name}}</h2>
    </div>
  `
})
export class HelloComponent {

  name: string;
  firstName  = 'John';

  constructor() {
    this.name = 'Angular';
  }
}
